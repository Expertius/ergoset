description: "Проверка типов — запуск tsc и исправление ошибок"
---

Run a full type check and fix any issues.

1. Run `npx tsc --noEmit` to check for TypeScript errors.
2. If there are errors, analyze each one and fix it.
3. After fixing, run `npx tsc --noEmit` again to verify all errors are resolved.
4. Report what was found and fixed.
